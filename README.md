## 구성 요소
* **html**: 문서의 루트요소, 모든 HTML요소 루트 요소내에 포함되어야 함
* **head**: 문서의 메타데이터 집합요소, 문서 제목, style, javascript 요소 포함
* **body**: 문서의 본문 요소

## 인터렉티브 요소
* **details**: 사용자의 선택에 따라 나타내거나 숨길 수있는 영역을 정의
* **summary**: 요소의 자식요소로 나타내거나 숨길 세부사항을 정의
* **dialog**: 사용자의 입력이나 응답을 요구하는 팝업 등을 마크업할 때
